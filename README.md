# Subtitles

A CLI to get plain text out of subtitles

## Supported formats

* `.srt`

## Usage

Use the binaries found on the release page. Look for the [latest release](https://github.com/chuckha/subtitles/releases/latest)!

```
subex file1.srt file2.srt file3.srt

# output files:
# file1.txt
# file2.txt
# file3.txt
```
